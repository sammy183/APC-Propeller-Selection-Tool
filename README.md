Steps to use:

1. Download the entire repo
2. Unzip 
3. Open USEME.py (numpy, matplotlib, and scipy need to be installed in the Python environment)
4. Modify Vinf (freestream velocity in m), RPM, Sw (wing area in m$^2$), CD (drag coefficient), and Plimit (Mechanical Power limit in W)
5. Enjoy!
